# RAI – Code Similarity Analyzer

RAI is a Visual Studio Code extension that detects similar or duplicate code fragments across your workspace using AI-based embeddings.

## Features

- Analyze code similarity using Hugging Face embeddings
- Detect duplicate or refactored code blocks
- View results in the sidebar
- Works offline (model cached locally)

## Commands

- **Analyze Similarity** – Run the similarity scan
- **Refresh** – Refresh analysis results
